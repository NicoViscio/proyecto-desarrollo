## Manual de Usuario

1. Ejecuta **dos instancias** de la misma aplicación: `chat_sockets_LNVF.exe`.
2. Introduce un **nombre de usuario** en ambas instancias.
3. Envía **mensajes**, **imágenes** o **emojis**.
